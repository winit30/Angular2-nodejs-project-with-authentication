# EITR Project
Easy IT Return Application


Technologies :

1. Node.js
2. Express.js
3. Sails.js
4. MongoDB

Prerequisites :  
Database should be created before starting the sailsjs project

Install :

Stept 1. 
git clone https://varenya@bitbucket.org/varenya/eitr-backend.git

cd eitr-backend

npm install

Step 2.
npm install nodemon --save

step 3.
start nodemon
